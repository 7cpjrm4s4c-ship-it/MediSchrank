⏰ Tägliche MHD-Benachrichtigung via Kurzbefehle

iOS-PWAs können keine Hintergrund-Benachrichtigungen senden. Die zuverlässige Lösung: ein iOS Kurzbefehl der täglich automatisch prüft und eine native Benachrichtigung schickt.

Kurzbefehl einrichten: 1. Kurzbefehle-App → + → Neuer Kurzbefehl
2. Aktion: „Datei abrufen" → iCloud Drive → MediSchrank → medischrank.json
3. Aktion: „Text aus Eingabe abrufen"
4. Aktion: „Wörterbuch abrufen" (JSON parsen)
5. Aktion: „Mitteilung anzeigen" → Text: „MHD prüfen!"
6. Tab Automation → + → Tageszeit → täglich 09:00 → Kurzbefehl auswählen
✓ In Zwischenablage — in Kurzbefehle-App einfügen
